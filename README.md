# Maison Dot

A full-stack real estate platform built with Laravel backend and Vue.js frontend.

## Project Structure

- **backend/** - Laravel API backend
- **frontend/** - Vue.js frontend application

## Features

- Property management
- User authentication
- Booking system
- Invoice management
- Subscription system

## Getting Started

### Backend Setup
```bash
cd backend
composer install
php artisan migrate
php artisan serve
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## Technologies Used

- **Backend**: Laravel, PHP
- **Frontend**: Vue.js, TypeScript, Tailwind CSS
- **Database**: MySQL/SQLite
